## 配置方法

### TVBox
设置 → 直播 → 添加订阅 → 粘贴M3U地址

### Televizo
设置 → 播放列表 → 添加 → 粘贴M3U地址

### KO影视
设置 → 直播源 → 导入 → 粘贴TXT地址

### IPTV Smarters Pro
Settings → Playlists → Add → 粘贴M3U地址

### TiviMate
设置 → 播放列表 → 添加 → 粘贴M3U地址

### Kodi (PVR IPTV Simple)
设置 → PVR → IPTV Simple Client → Remote Path → M3U URL
